¡Hola! Estás viendo Striped, una plantilla de sitio web HTML5 completamente receptiva diseñada por AJ para HTML5 UP. Incluye un diseño limpio y minimalista, estilos para todos los elementos básicos de la página (incluyendo citas, tablas y listas), una barra lateral reubicable (izquierda o derecha) y código HTML5/CSS3 diseñado para una personalización rápida y sencilla (consulta los comentarios del código para obtener detalles).

Striped se publica de forma gratuita bajo la licencia de Atribución de Creative Commons, así que siéntete libre de usarlo para proyectos personales o incluso comerciales, solo asegúrate de dar crédito a HTML5 UP por el diseño. Si te gusta lo que ves aquí, asegúrate de visitar HTML5 UP para ver más diseños geniales o seguirme en Twitter para conocer las novedades y actualizaciones.
